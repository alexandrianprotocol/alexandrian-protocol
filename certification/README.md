## Certification Artifacts

This folder contains machine‑generated artifacts produced by the certification pipeline.

- `m1-report.json` — structured output of `pnpm certify:m1` / `pnpm verify`, including stage results and hashes.

Human‑readable narratives and checklists live in:

- `docs/M1-CERTIFICATION-REPORT.md`
- `docs/milestones/M1-AUDIT-REPORT.md`
- `docs/milestones/M1-COMPLETION-CHECKLIST.md`

Re‑run certification using the instructions in `docs/VERIFY-M1.md`.
