## KB‑D — Knowledge Block D Artifact

This folder contains the IPFS‑anchored artifact and evidence bundle for **KB‑D**, a reference Knowledge Block used in M1 settlement proofs.

- `artifact.json` — canonical artifact bytes for KB‑D.
- `manifest.json`, `metadata.json`, `kb-record.json` — binding metadata and registry mapping.
- `docs/specification.md` — human‑readable specification for this KB.
- `evidence/*.json`, `evidence/logs.txt` — verification runs and settlement evidence.

These artifacts are part of the end‑to‑end M1 verification surface and are referenced from grant and verification docs.
