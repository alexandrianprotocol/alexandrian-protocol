## KB‑F — Knowledge Block F Artifact

This folder contains the IPFS‑anchored artifact and evidence bundle for **KB‑F**, a reference Knowledge Block used in M1.

- `artifact.json` — canonical artifact bytes for KB‑F.
- `manifest.json`, `metadata.json`, `kb-record.json` — binding metadata and registry mapping.
- `docs/specification.md` — human‑readable specification for this KB.
- `evidence/*.json`, `evidence/logs.txt` — verification runs and on‑chain publication evidence.

These artifacts are referenced from the root `README.md`, `docs/VERIFY-M1.md`, and the grant documentation under `docs/grants/`.
