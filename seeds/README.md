## Seeds

This folder contains **seed Knowledge Blocks** and artifacts used to initialize or demonstrate the protocol.

- `root/` — the canonical root seed for M1, with:
  - `envelope.json` — protocol‑valid envelope.
  - `artifact.json` — bound artifact bytes.
  - `artifactHash.txt` — integrity hash for the artifact.

Seeds are consumed by scripts and verification flows described in `docs/VERIFY-M1.md` and the M1 certification report.
