## Grants and External Write‑Ups

This folder contains grant applications, funding plans, and supporting material.

- `GRANT-COINBASE.md`, `GRANT-THE-GRAPH.md`, `GRANT-IPFS.md` — primary grant applications.
- `M2-FUNDING-EXECUTION-PLAN.md` — forward‑looking execution plan.
- `SUSTAINABILITY-MODEL.md` and `COST_SIMULATION.md` — economic model and cost simulations.
- `GAS-BENCHMARKS.md` — live gas costs and benchmarks.
- `LIVE-DEMO-ARTIFACT.md` — on‑chain settlement proof and royalty routing evidence.
- `IPFS-KB-F-CLI-DEMO.md` — IPFS/KB‑F CLI demonstration and artifacts.

Most of these documents are linked from the root `README.md` for reviewers; this index is the canonical place to browse them in‑repo.
