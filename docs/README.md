## Alexandrian Documentation

This folder is the top‑level hub for all protocol and product documentation.

- **Protocol spec**: consensus‑critical rules and invariants live under `protocol/`.
- **Milestones & certification**: audit and completion reports live under `milestones/`.
- **Grants & external write‑ups**: funding, gas benchmarks, and grant narratives live under `grants/`.
- **KB primitives**: foundational epistemic primitives live under `kb-primitives/`.
- **Verification**: see `VERIFY-M1.md` for the full M1 verification flow.

If you are new to the project:

1. Start with the root `README.md` for the high‑level overview.
2. Read `protocol/PROTOCOL-SPEC.md` for the canonical behavior.
3. Use `VERIFY-M1.md` to reproduce the M1 verification locally.
